基于黑金配色方案，以下是三个不同风格的布局设计方案，每个方案都强调**简洁、高效、易用**。

## 方案 C：极简命令式

**设计理念**：类似命令行，极简设计，适合键盘操作

```
┌─────────────────────────────────────────────────────────────────────┐
│                        ⚡ CLAUDE ROUTER                              │
│                                                                      │
│  > status                                                            │
│                                                                      │

│                                                                      │
│    ● RUNNING      OpenAI US-1      120ms      2.3 MB ↑↓            │
│                                                                      │
│    ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━    │
│                                                                      │
│                                                                      │
│  > configs                                                           │
│                                                                      │
│    生产环境                                                          │

│    └─ [4] Gemini Pro-1       ○ Offline                              │
│                                                                      │

└─────────────────────────────────────────────────────────────────────┘
```

**特点**：
- ✅ 类终端界面，极客风格
- ✅ 支持键盘快捷键和命令输入
- ✅ 界面简洁，信息密度高
- ✅ 适合开发者使用

**快捷键**：
- `Ctrl+K`: 聚焦命令输入
- `Ctrl+R`: 重启服务
- `Ctrl+1-9`: 快速切换配置
- `Ctrl+T`: 测试所有配置

## 配色方案（黑金风格）

所有方案统一使用以下配色：

### 主色调
- **背景**: `#000000` (纯黑) → `#0a0a0a` (深黑)
- **卡片**: `#1a1a1a` (灰黑) 带金色边框
- **金色主题**: `#FFD700` (金色) → `#FFA500` (橙金色)
- **强调**: `#B8860B` (暗金色)

### 状态色
- **运行中**: `#10B981` (绿色)
- **停止**: `#6B7280` (灰色)
- **错误**: `#EF4444` (红色)
- **警告**: `#F59E0B` (黄色)
- **离线**: `#4B5563` (深灰)

### 文字
- **主文字**: `#F3F4F6` (亮灰白)
- **次要文字**: `#9CA3AF` (中灰)
- **提示文字**: `#6B7280` (暗灰)

### 交互元素
- **按钮**: 黑底金边 + 悬停金色渐变
- **输入框**: 黑底金色边框
- **开关**: 金色激活态

## 推荐选择

### 🏆 **方案 A - 紧凑仪表盘** （推荐）
**理由**：
1. ✅ 最符合"简洁高效方便操作"的目标
2. ✅ 单屏完成所有核心操作，无需频繁切换页面
3. ✅ 配置列表清晰，状态直观
4. ✅ 保留了必要的高级功能入口
5. ✅ 适合 90% 的日常使用场景

### 其他方案适用场景
- **方案 B**：适合需要更多信息展示的场景，比如多分组、多配置管理
- **方案 C**：适合极客用户，喜欢键盘操作和命令行风格
